# t³

A minimal Rubik's cube speedsolving timer.

**https://pencils113.github.io/t3/**

## Usage

- **Hold spacebar** to arm (turns green), release to start
- **Press any key** to stop
- Data is stored locally in your browser

## Notes

- **Backup your data**: Export CSV periodically — browser data clearing will erase your solves
- **Internet required for scrambles**: Random-state scrambles load from cubing.js CDN; falls back to random-move scrambles if offline
